Function used to read the state of a GPIO pin.

Precondition
GPIO pin must be first initialized using the gpio_init_pin function.
Parameters
portGPIO pin port address (ex. &PORTA).
pinGPIO pin number (ex. PA1).
Returns
The current state of the GPIO pin: GPIO_PIN_LOW or GPIO_PIN_HIGH.

Example:

gpio_init_pin(&PORTA, PA1, GPIO_PIN_OUTPUT);
state = gpio_read_pin(&PORTA, PA1);
