Detailed Description

UART driver implementation.

To use the UART driver, include this header file as follows:

#include "uart.h"

Overview

The UART driver implements functions initialize, write and read from the UART.

Usage

To use the UART driver, the UART must be first initialized using the uart_init function.

The following code example initializes the UART and writes and receives data from the UART.

#include "uart.h"
char rxStr[20];
char txStr[] = "Hello!\n";
uart_send(fooStr);
uart_read(rxStr, sizeof(rxStr));
