Detailed Description

Sigfox Wisol driver implementation.

To use the Sigfox Wisol driver, include this header file as follows:

#include "sigfox_wisol.h"

Overview

The Sigfox Wisol driver implements funtions to write and read to the Sigfox Wisol module via ATT commands.

Usage

To use the Sigfox Wisol driver, the driver must be first initialized using the sigfox_wisol_init function.

The following code example initializes the module and reads the ID and PAC from the module.

#include "sigfox_wisol.h"
char str[20];
sigfox_wisol_init();
sigfox_wisol_get_id(str, sizeof(str));
sigfox_wisol_get_pac(str, sizeof(str));
